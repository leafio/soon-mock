# Data

有2类型的数据存储：JSON键值对 `{}` , JSON数组 `[]`

### JSON数组

- 如下图，JSON数组会显示在表格中
- 可以 `编辑` or `删除` 单条数据
- 可以批量添加新数据
- 可以从 `JSON 数组` 文本 或 `xlsx` 文件 批量导入数据。
- 可以清空数据，使之变为空数组 `[]`
